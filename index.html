(주)인시즌 - IoT 스마트공방 프로젝트

2021 (회사 프로젝트)

- 온도센서, 습도 센서를 이용하여 데이터를 받아오고 해당 데이터를 DB에 저장, DB에서 가져온 데이터를 차트와 테이블로 시각화하는 웹사이트 작성
- 기능1. 장소에 따른 온도와 습도 데이터를 차트로 표현 (Bootstrap Table) - 같은 장소에 있는 데이터들의 평균값
- 기능2. 센서 별 데이터 값 차트로 보여주기
- 기능3. 지정한 기간의 데이터 차트 및 테이블 보여주기 (History) - 센서가 off 됐을 경우 null값이 입력되도록 하여 한눈에 확인 가능하도록 함.
- 기능4. 센서 등록. 기본 키 값으로 센서마다 시리얼 넘버가 할당됨. 시리얼 넘버로 센서에서 보내는 데이터 구분 가능.
- 기능5. 관리자 등록. 관리자를 등록할 때 온도, 습도의 최댓값을 입력하여 해당 수치가 넘어가면 관리자 등록할 때 입력한 전화번호로 카카오 알림톡 전송과 관리자의 이메일 주소로 이메일 전송.
